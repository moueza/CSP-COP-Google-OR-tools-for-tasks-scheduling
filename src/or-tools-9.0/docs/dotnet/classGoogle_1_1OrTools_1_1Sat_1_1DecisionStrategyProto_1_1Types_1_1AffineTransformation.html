DecisionStrategyProto.Types.AffineTransformation

Detailed Description

Advanced usage.

Some of the variable listed above may have been transformed by the presolve so this is needed to properly follow the given selection strategy. Instead of using a value X for variables[index], we will use positive_coeff * X + offset instead.
