◆ ImprovementRateCoefficient

double ImprovementRateCoefficient
getset

Parameter that regulates exchange rate between objective improvement and number of neighbors spent.

The smaller the value, the sooner the limit stops the search. Must be positive.

◆ ImprovementRateSolutionsDistance

int ImprovementRateSolutionsDistance
getset

Parameter that specifies the distance between improvements taken into consideration for calculating the improvement rate.

Example: For 5 objective improvements = (10, 8, 6, 4, 2), and the solutions_distance parameter of 2, then the improvement_rate will be computed for (10, 6), (8, 4), and (6, 2).
